# 📊 E-commerce Purchase Analysis

## 📌 Project Overview

This project explores an e-commerce purchases dataset using **Python** and **Pandas**. The objective is to answer business-oriented questions through exploratory data analysis (EDA) and practice common data manipulation techniques.

---

# 🎯 Objectives

The analysis aims to:

- Explore the dataset structure
- Perform descriptive statistics
- Analyze customer purchasing behavior
- Examine payment methods
- Investigate customer demographics
- Practice data filtering and aggregation with Pandas

---

# 🗂️ Dataset

The dataset contains customer purchase information, including:

- Customer Name
- Email
- Language
- Job
- Purchase Price
- Credit Card Provider
- Credit Card Number
- Credit Card Expiration Date
- IP Address
- Purchase Time (AM/PM)

---

# 🔍 Analysis

The notebook answers several analytical questions, including:

## Purchase Analysis

- Average purchase price
- Highest purchase price
- Lowest purchase price

## Customer Analysis

- Number of French-speaking customers
- Customers working as Engineers
- Customer lookup by IP address
- Customer lookup by credit card number

## Payment Analysis

- Most common credit card providers
- Customers spending more than a specified amount
- Credit cards expiring in a given year

## Purchase Behavior

- Number of AM vs PM purchases
- Most common email providers

---

# 💡 Key Takeaways

Through this analysis, the project demonstrates how customer transaction data can be explored to:

- Understand purchasing behavior
- Analyze customer demographics
- Explore payment preferences
- Practice answering business questions using Pandas
